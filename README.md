# calculator
一个基于web的通用计算器程序

[点击预览]: (http://suyunsoft.com/calculator/calcuator.html)

计算器快捷键如下：

| 快捷键                 | 功能                    |
| :------------------ | :-------------------- |
| 小键盘数字键              | 输入对应数字                |
| 小键盘 `+` `-` `*` `/` | 对应数学符号`+` `-` `*` `/` |
| `BackSpace`         | 删除单个数字                |
| `Delete`            | 删除整行                  |
| `Enter`             | 计算结果                  |
| `Esc`               | 归零并退出计算器              |
| `F2`                | 调用计算器                 |

可根据实际情况添加计算器“容器”进行二次开发
